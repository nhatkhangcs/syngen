About Synthetic Generator

A comprehensive Python library for generating high-quality synthetic data

Key Features

Multiple Distributions

Support for Normal, Uniform, Exponential, Gamma, Beta, Weibull, Poisson, Binomial, and more.

Correlations & Dependencies

Define relationships between variables and generate data with realistic correlations.

Privacy Protection

Built-in privacy features including anonymization and differential privacy support.

Technical Specifications
  • Python Version: 3.8+
  • Core Dependencies: pandas, pydantic, numpy, scipy
  • Web UI: Flask, Bootstrap 5, JavaScript
  • License: MIT License
  • Version: 0.1.7
